YouTube Player (Embed in Your Website)

Paste a YouTube link → this page converts it to an /embed/VIDEO_ID URL and plays it in an iframe.

Single file • HTML + CSS + JS
Status: Paste a YouTube link and click Play.

Autoplay

Loads the video and tries to start automatically.

Mute (recommended with autoplay)

Starts muted (some browsers block autoplay with sound).

Privacy-enhanced mode

Use youtube-nocookie.com for the embed host.

Show controls

Show/hide the player controls.

Tip: If a video does not play, the uploader may have disabled embedding or your network may restrict YouTube domains.
Embed URL format and player parameters are based on the official YouTube embedded player documentation. [1](https://developers.google.com/youtube/player_parameters)
